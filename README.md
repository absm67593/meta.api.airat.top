## 🖥️ What this app does

meta.api.airat.top is a small web tool that reads the metadata from any public webpage. You give it a URL, and it returns useful page info in a clean format.

It can show:

- Page title
- Page description
- Canonical URL
- Robots data
- Open Graph tags
- Twitter card data
- Page language
- Page charset

You can view the result as:

- JSON
- Text
- YAML
- XML

This makes it useful when you want to check how a page looks to search engines and social sites.

## 🔍 What metadata you can expect

When you enter a URL, the app checks the page source and looks for common metadata fields.

It can return:

- `<title>`
- `<meta name="description">`
- `<link rel="canonical">`
- `<meta name="robots">`
- `<meta property="og:...">`
- `<meta name="twitter:...">`
- `lang`
- `charset`

This helps you see how the page may appear in search results, link previews, and social shares.
